API Reference

The Giant Swarm API is a simple, mostly RESTful JSON based API, which can be accessed over an encrypted HTTPS connection. The API is used by our swarm CLI and can be incorporated into your software wherever you require programmatic control over your infrastructure running on Giant Swarm.

We currently offer two versions of our API. We recommend to use API v2 wherever possible, since API v1 is to be phased out at some point. Also, watch the API changelog for new v2 methods being added, superseding the according v1 methods.

Account owners will get notice when a roadmap/timeline for deactivation the v1 API is decided on.

API v2 Reference

The latest API version with asynchronous tasks. Doesn’t cover the full spectrum of functionality yet, so in most cases you will have to use it in combination with API v1.

API v1 Reference

Our first API with the complete coverage of functionality. Some methods are deprecated, since there are now improved versions available in the API v2.
