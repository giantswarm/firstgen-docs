Giant Swarm on GitHub

We maintain a set of examples hosted on our GitHub account:

Hello World

  • giantswarm/helloworld - a hello world example using a pre-built image.

Language Specific

  • giantswarm/giantswarm-firstapp-go - an example in Go.
  • giantswarm/giantswarm-firstapp-nodejs - an example in NodeJS.
  • giantswarm/giantswarm-firstapp-php - an example in PHP.
  • giantswarm/giantswarm-firstapp-python - an example in Python.
  • giantswarm/giantswarm-firstapp-ruby - an example in Ruby.
  • giantswarm/sample_app_rails_4 - an example in Ruby on Rails.
  • giantswarm/giantswarm-firstapp-java - an example in Java.

Application Specific

  • giantswarm/giantswarm-mysql-archiver - an example showing methods to back up MySQL.
  • giantswarm/swarm-wercker - a continous deployment example using Wercker.
  • giantswarm/swarm-ghost - an example for deploying a Ghost blog.
  • giantswarm/swarm-ogar - NodeJS w/websockets game server example.

Using the APIs

  • giantswarm/swarm-cubism - an example graphing service stats using the APIs.

The CLI and Related Repositories

  • giantswarm/cli - The swarm command line interface utility
  • giantswarm/client-go - Our own Go client library for the use of our API
Need help, got feedback?

Join us at #giantswarm on IRC or send us an email to support@giantswarm.io. And of course, we welcome your pull requests!
