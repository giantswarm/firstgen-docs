swarm scaledown: Scale Down Component

The swarm scaledown command is used to decrease the number of component instances.

Command Syntax

The swarm scaledown command is called using a component name and the number of instances to remove from the component as arguments. If a number of instances is not given, the number of instances is scaled down by one (1) instance.

swarm scaledown <service/component> <num_to_remove>

Note: The swarm scaledown command has no effect when a component has a single running instance.

Example with Response

$ swarm scaledown currentweather/flask 2
Scaling down component currentweather/flask by 2...
