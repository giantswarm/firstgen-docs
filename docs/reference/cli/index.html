The swarm CLI

The swarm CLI is our command line tool used to deploy, query and control your Giant Swarm services.

Getting Started

Please refer to the following guides to get started using Giant Swarm’s swarm client:

  • Installation and Configuration - instructions on how to set up the CLI on OS X and Linux.
  • Global Command Line Options - options for working with CLI commands.
  • Release Notes - information about changes in the CLI releases.

Command Reference

Click on the links below for more details on a particular command usage:

Command Description
swarm cat Show the definition of a service
swarm completion Setup CLI completion
swarm create Create a new service using a JSON file
swarm delete Delete a service
swarm env Manage environments
swarm exec Execute commands on a component instance
swarm info Get basic information on context and environment
swarm login Sign in to a Giant Swarm account
swarm logout Log out from an account
swarm logs Print logs from a component instance
swarm ls List all services in the current enviroment
swarm migrate Migrate a service definition file
swarm org Manage organizations
swarm scaledown Reduce number of instances running a component
swarm scaleup Increase number of instances running a component
swarm start Start a service or component
swarm stats Print basic statistics for service, component or instance
swarm status Show current status of a service, component or instance
swarm stop Stop a service or component
swarm up Create and start a new service
swarm update Update a service definition or component
swarm user Create and modify users
swarm version Print the CLI version
